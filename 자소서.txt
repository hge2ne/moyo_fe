최신 웹 기술과 빠르게 발전하는 AI 기반 도구를 이용한
직관적이면서도 강렬한 첫 인상을 남기는 인터페이스를 구축하여,
서비스와 깊이 연결되는 듯한 몰입감 높은 UX/UI를 제공하는 프론트엔드 개발자가 되고싶습니다. 

과거 Figma와 React를 활용한 프로젝트에서 Cursor IDE, 피그마 mcp 와 같은 AI 도구로 빠르게 웹 페이지를 구현해보았습니다. 
구현했지만, AI가 생성한 사용자 입력 처리 코드에 XSS 취약점이 있다는 것을 뒤늦게 발견했습니다. 
또한 불필요하게 복잡하거나 성능이 떨어지는 코드를 그대로 사용해 로딩 속도가 현저히 느려지는 문제도 겪었습니다. 
SSAFY에서 웹 보안과 성능 최적화의 기초를 탄탄히 다지고, 
AI 도구를 활용하되 생성된 코드를 실무에서 개선할 수 있는 역량을 기르고 싶습니다. 
특히 실전 프로젝트를 통해 보안과 성능을 모두 고려한 고품질 코드를 작성하는 법을 체득하여, 사용자가 안전하고 쾌적하게 서비스를 이용할 수 있는 인터페이스를 구축하는 개발자로 성장하고 싶습니다."

Cursor IDE와 같은 AI 도구의 도움을 받아 빠르고 효율적으로 고품질의 재사용 가능한 컴포넌트를 구축하고 싶습니다. 
이를 통해 팀 프로젝트 시 프론트엔드 개발의 생산성을 극대화하고, 반복 작업을 줄여 사용자 경험 개선이라는 본질적인 문제에 더 집중하고 싶습니다. 
궁극적으로는 LLM API를 활용해 사용자의 맥락을 이해하고 동적으로 최적의 UI 컴포넌트 조합을 제공하는 개인화된 서비스를 만드는 데 기여하고 싶습니다. 
팀 프로젝트에서는 UI 컴포넌트 라이브러리 구축 및 관리, 그리고 개발 표준을 수립하여 팀 전체의 개발 효율성을 높이는 역할을 하고 싶습니다.
